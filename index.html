John's Coding Page

John is a professional teacher/trainer that also “self”-taught himself coding with a little help from books, the internet, and some friends. First things first, coding is not easy if you know nothing about it. However, even after a few days someone that knew nothing about coding can create a simple webpage for themselves. The biggest problem I came across when trying to find solutions to my problems was that I was speaking a different language. My first language is English, and I was trying to learn from people that spoke a different language, code language. That being said, the first thing everyone trying to learn code should do is learn the language. Once you know that language, searching for answers and troubleshooting problems becomes immensely easier. Below, you will find some meaning to some words that you have heard before.
